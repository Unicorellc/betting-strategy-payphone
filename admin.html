🔑 Configurar acceso a GitHub

Para editar y guardar los datos del dashboard necesitas un token de acceso personal de GitHub.

Como crear tu token

  1. Ve a github.com/settings/tokens (Fine-grained tokens)
  2. Haz clic en "Generate new token"
  3. Nombre: Betting Dashboard Admin
  4. Expiracion: elige 90 dias o mas
  5. En "Repository access" selecciona "Only select repositories" y elige Unicorellc/betting-strategy-payphone
  6. En "Permissions" → "Repository permissions" → Contents: selecciona Read and write
  7. Clic en "Generate token" y copialo
Token invalido o sin permisos. Verifica que tenga acceso al repositorio.
